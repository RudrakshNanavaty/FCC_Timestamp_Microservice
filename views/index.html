Timestamp Microservice

Example Usage:

  • Return current GMT date and time in Unix and UTC formats:
    [project url]/api
  • Return UTC representation and Unix timestamp of given date and time:
    [project url]/api/2015-12-25
  • Return UTC representation and Unix timestamp of given timestamp:
    [project url]/api/1451001600000

Example Output:

{ "unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT" }
